About Research Work Experience CV

I study uncertainty quantification and decision-making under distribution shift—methods that are principled yet practical.

Current Research

  • Nonparametric Confidence Intervals using the Cheap Bootstrap
    Ongoing work with Prof. Henry Lam

    Lightweight bootstrap that yields asymptotically exact CIs for bagged predictors with a small, fixed number of resamples as bag size grows; applicable to Random Forests and bagged SAA.

    Link: Add link
  • Superiority of Naive Optimization via Stochastic Dominance
    Ongoing work with Prof. Henry Lam

    Shows when ERM can outperform DRO/regularization in minimizing worst-case regret under joint data and distributional uncertainty via generalized symmetry arguments.

    Link: Add link
  • Estimation of Rare-Event Probabilities using Extreme Value Theory
    Ongoing work with Prof. Henry Lam

    Estimates tail probabilities using GPD as a model-agnostic alternative to importance sampling for rare-event simulation.

    Link: Add link
  • Fantasy-Sports User Behavior Modeling
    Ongoing collaboration with Dream Sports

    Identifies platform actions (promotions/incentives) that causally impact user engagement at production scale.
